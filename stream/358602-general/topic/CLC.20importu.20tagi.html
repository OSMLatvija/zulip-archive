HellMap (Jun 28 2025 at 09:32):

Yes, CLC stands for "CORINE Land Cover" and was added when forests were being imported. The tag is a bit misused, it should have been in source. But also, it shouldn't be added to begin with because it's the changeset that defines the source. Almost every single forest has been edited since, so these values are out of date - the forest no longer matches CLC data. I personally remove these whenever I modify the geometry. At some point we should just agree to finally remove them all in bulk.
