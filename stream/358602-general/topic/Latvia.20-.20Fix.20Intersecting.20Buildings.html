Hello Latvia community,

I would like to inform you that we have created a new MapRoulette challenge named Latvia - Fix Intersecting Buildings, we’re focusing on finding and fixing cases where buildings overlap on the same layer. When buildings intersect like this, it can make maps look cluttered and harder to read, which can be confusing. By fixing these overlaps, we’ll help make the map clearer and more accurate.

How Tasks Were Created
We generated these tasks by comparing building shapes in OSM to spot overlaps. While most tasks should highlight true intersections, some may be false positives. If that’s the case, feel free to mark them as "Not an Issue" and move on.

Thank you and have a wonderful day,

HellMap (Oct 31 2024 at 11:09):

It should be worth mentioning in the task description that Latvia has a cadaster layer available with building footprints that range between accurate and very approximate. iD and JOSM have it in the imagery layers. It can often be used to determine building extents and make corrections. Not always possible, but in many instances much better than guessing or using aerial only.
