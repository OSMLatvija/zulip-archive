HellMap (Mar 11 2026 at 13:27):

So the way I've tagged them. In Latvia, where there is no infrastructure, it should be crossing=no. It's not a crossing at all - it's illegal. It just so happens that a path is there and technically people can and do cross, so we represent on-the-ground reality of there being a "crossing".

crossing=informal implies it's not illegal, as far as I understand. But the tag hasn't been formally discussed, so it's possible we might eventually do crossing=informal + access=no. I don't think in Latvia there is an instance of crossing=informal - it's either an actual legal crossing or it's not at all. Because if a crossing is not intended (i.e. informal), then in Latvia it's de jure illegal.

Routers currently suck either way.
